Система управления людьми с персистентностью

Возможности системы:

  • Валидация данных: проверка корректности фамилии, имени, отчества, телефона, email
  • Дедупликация: автоматическое объединение записей в группы по правилам
  • Персистентность: возможность просмотра состояния на любой момент времени
  • REST API: полный набор endpoints для работы с данными

API документация

REST API endpoints для интеграции

API Endpoints:

Работа с людьми:

Метод URL Описание Параметры
POST /api/persons/ Создание нового человека с автодедупликацией JSON: last_name, first_name, birth_date, gender, address, phone?, email?
GET /api/persons/list/ Получение списка всех текущих людей Нет
GET /api/persons/search/ Витрина поиска - дедуплицированная информация last_name, first_name, middle_name, address, phone, email, limit, offset
GET /api/persons/{group_id}/as-of/ Состояние человека на момент времени timestamp (ISO 8601)

Работа с адресами (DaData):

Метод URL Описание Параметры
GET /api/address/suggestions/ Подсказки адресов при вводе query (строка поиска), count (количество)
POST /api/address/clean/ Стандартизация и проверка адреса JSON: address
GET /api/address/geocode/ Получение координат по адресу address (строка адреса)

Персистентность (Git-like versioning):

Метод URL Описание Параметры
GET /api/persistency/groups/ Список всех групп Нет
GET /api/persistency/groups/{group_id}/history/ История изменений группы по ID limit (количество записей)
GET /api/persistency/groups/{group_id}/at-time/ Состояние группы на момент времени по ID timestamp (ISO 8601)
GET /api/persistency/groups/{group_name}/history/ История изменений группы по имени limit (количество записей)
GET /api/persistency/groups/{group_name}/at-time/ Состояние группы на момент времени по имени timestamp (ISO 8601)
GET /api/persistency/groups/{group_name}/compare/ Сравнение состояний группы timestamp1, timestamp2 (ISO 8601)
POST /api/persistency/groups/{group_name}/members/ Добавление участника в группу JSON с данными участника
DELETE /api/persistency/groups/{group_name}/members/{person_id}/ Удаление участника из группы Нет
GET /api/persistency/changesets/ Список всех changesets (коммитов) limit (количество записей)
GET /api/persistency/changesets/{changeset_id}/ Детали changeset Нет
GET /api/persistency/persons/{person_id}/history/ История изменений человека Нет

Примеры использования:

Поиск в витрине:
# Поиск по фамилии
GET /api/persons/search/?last_name=Иван

# Поиск по фамилии и имени  
GET /api/persons/search/?last_name=Петров&first_name=Иван

# Поиск по адресу
GET /api/persons/search/?address=Москва

# Поиск по телефону
GET /api/persons/search/?phone=89154233426
Персистентность:
# История группы
GET /api/persistency/groups/1/history/

# Состояние на момент времени
GET /api/persistency/groups/1/at-time/?timestamp=2025-11-06T18:40:00Z

# Список всех коммитов
GET /api/persistency/changesets/

# История человека
GET /api/persistency/persons/123/history/
